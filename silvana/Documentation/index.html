Installing The Theme


Log into your site via FTP . (if you don't have any FTP program you can use FileZilla , it's free).

 

The steps taken.

  • Unzip the file downloaded from ThemeForest.
  • Upload Fresh_installation.zip (in folder "For New Fresh Drupal Installation") to your domain directory and unzip it.
  • Import file "database/sample_data.sql" or "database/sample_data.zip" to phpMyadmin (http://localhost/phpmyadmin/ of http://your_website.com/phpmyadmin)
  • Edit file "sites/default/settings.php".

  • Done! Now, go to your website.
  • Username and Password:
    - User: "admin"
    - Password: "admin"
